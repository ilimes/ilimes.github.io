### ✔️ 문제

<https://school.programmers.co.kr/learn/courses/30/lessons/87389>

### ✔️ 코드

```jsx
function solution(n) {
  for (let i = 0; i < n; i++) {
    if (n % i === 1) {
      return i;
    }
  }
}
```

### ✔️ 풀이

단순하게 for 반복문을 사용하여 해결
