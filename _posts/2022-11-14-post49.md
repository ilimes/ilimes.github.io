### ✔️ 코드

```jsx
function solution(price, money, count) {
  let answer = 0;
  for (let i = 1; i < count + 1; i++) {
    answer += price * i;
  }

  answer > money ? (answer = answer - money) : (answer = 0);
  return answer;
}
```

### ✔️ 풀이

이용 횟수 만큼 반복문 돌려주고 더한 값 구한 뒤 money와 비교하여 answer값 return
