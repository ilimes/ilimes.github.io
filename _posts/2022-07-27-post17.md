피그마로 작업 할 때 조금이라도 작업 효율을 높이고 싶다면 단축키를 활용하는 것이 좋습니다. 요즘 피그마 작업을 하면서 단축키를 활용하려고 하는데 자주 사용하지 않으면 자꾸만 까먹게 되더라고요. 이번 포스팅은 제가 다시 보기 위해서 작성하게 된 피그마 단축키 모음입니다 😊

### 전체 피그마 단축키 보는 법

- `Ctrl` + `Shift` + `?`

### 그리드

- 레이아웃 그리드 보기 : `Ctrl` + `Shift` + `4`

### 컴포넌트

- 컴포넌트 생성 : `Ctrl` + `Alt` + `K`
- 인스턴스 분리 : `Ctrl` + `Alt` + `B`
- 오토레이아웃 생성 : `Shift` + `A`
- 오토레이아웃 제거 : `Alt` + `Shift` + `A`
- 컴포넌트 찾기/삽입 : `Ctrl` + `Alt` + `B`
- 컴포넌트명 한번에 수정 : `Ctrl` + `R`
- 만들어 놓은 객체 컴포넌트화 : `Ctrl` + `Alt` + `G`

### 정리하기

- 앞으로 가져오기 : `Ctrl` + `]`
- 뒤로 보내기 : `Ctrl` + `[`
- 맨 앞으로 가져오기 : `Ctrl` + `Shift` + `]`
- 맨 뒤로 보내기 : `Ctrl` + `Shift` + `[`

### 모양 변형

- 가로로 뒤집기 : `Shift` + `V`
- 세로로 뒤집기 : `Shift` + `H`
- 아웃라인 스트로크 : `Ctrl` + `Shift` + `O`

### 레이어

- 전체화면에서 레이어 선택 : `Ctrl` + `Alt` 누른 상태에서 객체 클릭
- 모든 레이어 닫기 : `Alt` + `L`

### 확대 및 축소

- 화면에 캔버스 핏 : `Shift` + `1`
- 선택한 개체 확대 : `Shift` + `2`
- 배율 100%로 맞추기 : `Shift` + `0`
- 확대 : `+` 또는 `Z`를 누른 채 클릭
- 축소 : `-` 또는 `Z`와 `Alt`키를 누른 채 클릭

### 불투명도 설정

- `0` ~ `9` 숫자키
  - 특정 요소를 선택한 상태에서 숫자키를 사용하면 빠르게 불투명도 설정을 할 수 있습니다.  
    보다 세밀하게 조정하고 싶다면 두 숫자를 연속으로 빠르게 입력하면 됩니다.
